## 💻 Getting Started <a id="getting-started"></a>


To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

> git clone https://github.com/DavidEspinoG/microverse_react_practice.git
> (you can open it via live server in VS)

### Setup

Clone this repository to your desired folder:

> git clone https://github.com/DavidEspinoG/microverse_react_practice.git

### Install

Install this project with:

npm install

### Usage

To run the project just open it in VS and run liveServer


### Run tests

To run tests, run the following command:

npm run fix

### Deployment

You can deploy this project using:

use chosen hosting service or github pages
